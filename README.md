# onyxblds
automated builds of box64, LTS UML kernels, and other tools for onyx

**(every new build comes at ~4:00-5:00 monday UTC)**

## where to get
commands:
```bash
# box64
curl -LO https://github.com/arozoid/onyxblds/releases/latest/download/box64-arm64
# UML (user mode linux) kernel 6.12 (arm64)
curl -LO https://github.com/arozoid/onyxblds/releases/latest/download/uml-arm64-lts6
# UML (user mode linux) kernel 5.15 (arm64)
curl -LO https://github.com/arozoid/onyxblds/releases/latest/download/uml-arm64-lts5
# UML (user mode linux) kernel 6.12 (x86_64)
curl -LO https://github.com/arozoid/onyxblds/releases/latest/download/uml-x86_64-lts6
# UML (user mode linux) kernel 5.15 (x86_64)
curl -LO https://github.com/arozoid/onyxblds/releases/latest/download/uml-x86_64-lts5
```
